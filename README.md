# V2I for WW

一个从视频生成图片序列的小工具

## 使用方法

将视频拖到软件上运行即可，支持多个视频处理

**会在软件同级下按照视频名生成输出目录**

## 参数

默认值：

| 参数     | 值        |
| ------ |:--------:|
| 输出分辨率  | 1280x760 |
| 输出帧率   | 30fps    |
| 输出时长   | 15s      |
| 输出图片数  | 450      |
| 输出图片格式 | jpg      |

注：暂时还无法自定义参数
